Colors

Color palettes define the visual identity of ORCA. The primary color is Emerald (#10b981), with Slate/Zinc grays for surfaces. Click on any color to copy its CSS variable.

Primary Color (Emerald)

The primary color is used for interactive elements, links, buttons, and primary actions. Emerald provides a fresh, modern, and accessible color that works well in both light and dark modes.

Primary (Semantic Aliases)

Use these semantic variables instead of the raw emerald values for better maintainability. If the brand color changes, only the mapping needs to update.

Surface Colors (Secondary)

Surface colors define backgrounds, borders, and neutral elements. These are the "secondary" colors that complement the primary Emerald.

Slate (Light Mode Base)

Used for surfaces in light mode. Slightly cool gray with blue undertones.

Zinc (Dark Mode Base)

Pure neutral gray used for surfaces in dark mode.

Gray (Alternative Neutral)

Standard gray scale for general use.

Surface (Semantic Aliases)

Use these semantic surface variables for automatic light/dark mode support. The values change based on the current theme.

Key Semantic Tokens:

--surface-card Card backgrounds
--surface-border Border colors
--surface-hover Hover states
--text-color Primary text
--text-color-secondary Secondary text

Semantic State Colors

These colors communicate meaning: success, information, warning, danger, and help states.

Extended Palette

Usage

Use CSS custom properties to apply colors consistently across the application.

Accessibility

WCAG AA Compliance

All text colors meet minimum contrast ratios (4.5:1 for normal text, 3:1 for large text). Emerald-500 on white has a 3.4:1 ratio - suitable for large text and icons.

Dark Mode Support

Colors are automatically adjusted for dark mode. Semantic tokens like --surface-card and --text-color switch values automatically based on theme.

Color Blindness

State colors use sufficient lightness contrast to remain distinguishable for users with color vision deficiencies. Never rely on color alone to convey information.
