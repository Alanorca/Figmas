Menu Item Structure

Common properties for menu items across all menu components.

Property Type Description
label string Text to display
icon string Icon class (PrimeIcons)
routerLink string Router navigation path
command function Callback function on click
items MenuItem[] Sub-menu items
separator boolean Renders a separator line
badge string Badge value to display
disabled boolean Disables the menu item
visible boolean Controls visibility
