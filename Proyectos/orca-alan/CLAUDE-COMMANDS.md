# Claude Commands - Referencia Rapida

## Comandos Disponibles

### Design System Guardian (DSG)
**Invocacion**: `agente DSG` | `/dsg` | `/design-system-guardian`

Guardian del Design System que audita y corrige problemas de temas y estilos.

**Subcomandos**:
- `agente DSG` - Inicia auditoria interactiva
- `agente DSG audit [pagina]` - Audita pagina especifica
- `agente DSG fix` - Corrige bugs encontrados
- `agente DSG learn` - Actualiza base de conocimiento

**Archivos**: `.claude/commands/design-system-guardian.md`
